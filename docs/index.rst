=======================
Anitya Messaging Schema
=======================

This is a documentation for Anitya Fedora Messaging Schema.

It's goal is to provide unified messaging schema for
`Anitya project <https://github.com/fedora-infra/anitya>`_. 
Schema is defined for `fedora messaging
<https://fedora-messaging.readthedocs.io/en/stable/>`_.

:Github page: https://github.com/fedora-infra/anitya-messages
